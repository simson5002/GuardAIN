# 🛡 GuardAIN – Agentic AI Browser Security System

GuardAIN (Guard Artificial INtelligence) is a privacy-first, agentic AI-powered browser extension designed to detect and prevent phishing and unsafe websites in real time.

This project is developed as part of the **B.Tech CSE Mini Project (Semester 6)** under APJAKTU 2019 Scheme, following proper **Software Development Life Cycle (SDLC)** practices.

---

## 🚀 Project Objective

To build a **lightweight, privacy-preserving browser security system** that:
- Detects phishing and suspicious websites
- Uses **on-device AI (no cloud dependency)**
- Applies **agentic decision-making** (observe → analyze → decide → act)
- Provides **clear explanations to users**

---

## 🧠 Key Features

- 🔍 Real-time URL analysis
- 🤖 Machine Learning-based phishing detection
- 🧩 Agentic AI decision engine (LangChain-inspired)
- 🔐 100% on-device processing (privacy-first)
- ⚡ Fast and lightweight (Manifest V3 compliant)
- 📢 User warnings and blocking system
